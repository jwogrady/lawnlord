## What v0.1.0 is

The already-working, deterministic legal court-record **exploder**: one source ZIP of court
PDFs → a five-level corpus (`archive → submission → document → section → page`) with full
provenance. No new features beyond what already existed — this release was documentation and
release hygiene only.
